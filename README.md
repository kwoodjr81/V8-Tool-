# 🚀 ProfitAI v8 — AI Dropshipping Profit Analyzer

ProfitAI v8 is a lightweight SaaS tool that helps dropshippers and ecommerce sellers instantly calculate product profitability and get an AI-style verdict on whether a product is worth selling.

It combines:

- Profit calculation

- Margin analysis

- Ad cost breakdown

- AI-style scoring system

- Winning / Test / Bad product prediction

## 🧠 How It Works

1. Enter product cost

2. Enter shipping cost

3. Enter selling price

4. Enter ad cost per sale

5. Click “Analyze Product”

6. Get instant profit + AI score + verdict
